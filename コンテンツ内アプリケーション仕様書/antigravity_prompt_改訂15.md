## 対象ファイル

- `車椅子_自走式介助式選定/車椅子タイプ診断アプリ.html`
- `コンテンツ作成/コンテンツ案/車椅子タイプ診断アプリ.html`

修正対象は `renderResult()` 関数内、結果画面のHTML組み立て部分です。

## 背景（なぜこの修正が必要か）

現状、以下の注記は「自走式（isSelf === true）」の結果画面にしか表示されません。

```js
if(isSelf){
  html += '<div class="result-note warn"><b>①サイズ・設置場所のご確認</b>ご購入前に、実際に車椅子を使う場所（玄関・廊下の幅など）や、車に載せる場合はトランクのサイズもあわせてご確認ください。</div>';
}
```

しかし、屋内の廊下幅・転回スペースや車のトランクサイズは、自走式・介助式のどちらであっても物理的に同じ車椅子を通す・積むという話であり、介助式の結果でも同様に問題になり得ます。この注記が自走式限定になっているのは設計の抜けであり、MD仕様書（`車椅子タイプ診断ロジック_仕様書.md`）の改訂15で、この注記を「屋内での通行・転回」と「車のトランクサイズ」の2つに分割し、両方とも全結果共通表示に変更しました。一方、「②今後の身体状況の変化」の注記（進行性疾患などで将来自走式から介助式へ切り替える可能性がある、という内容）は自走式という選択自体に伴うリスクの話であり介助式には当てはまらないため、これは従来どおり自走式限定のまま維持します。

## 具体的な変更内容

### 1. 削除するコード

`renderResult()` 関数内の、次の自走式限定ブロックを削除します（このブロックは商品カード表示 `<div class="products">...` より前、`isSelf` 分岐の中にあります）。

```js
if(isSelf){
  html += '<div class="result-note warn"><b>①サイズ・設置場所のご確認</b>ご購入前に、実際に車椅子を使う場所（玄関・廊下の幅など）や、車に載せる場合はトランクのサイズもあわせてご確認ください。</div>';
}
```

### 2. 追加するコード

削除した箇所と同じ位置（商品カード表示ブロック `html += '<div class="products">';` の直前）に、`isSelf`条件をつけず、全結果共通で表示する形で以下を追加します。

```js
html += '<div class="result-note warn"><b>屋内での通行・転回について</b>ご自宅で車椅子を使う場所（廊下・出入り口の幅など）を通れるか、方向転換できるスペースがあるかを、ご購入前にご確認ください。</div>';
html += '<div class="result-note warn"><b>車のトランクサイズについて</b>車に載せて外出する場合は、車種・トランクのサイズと車椅子（折りたたみ時のサイズ）が合うかを事前にご確認ください。</div>';
```

### 3. 変更しないもの（重要）

以下は改訂15の対象外です。誤って削除・変更しないでください。

- 「②今後の身体状況の変化について」の注記ブロック（`isSelf`限定のまま維持）：
  ```js
  if(isSelf){
    html += '<div class="result-note warn"><b>②今後の身体状況の変化について</b>今のところ自走式が合っていても、進行性の病気などで今後身体状況が変わる可能性がある場合は、将来的に介助式への切り替えが必要になることがあります。ご不安な場合は、介助式もあわせてご検討いただくか、福祉用具専門相談員にご相談ください。</div>';
  }
  ```
  ※新しい「屋内での通行・転回について」注記と内容が近く見えますが、こちらは自走式限定のまま残してください（自走式という選択そのものに伴う将来リスクの注記であり、屋内通行の物理的な確認とは別の話のため）。

- `state.reason === "II-no"` のときに表示される、屋内の取り回しに関する既存の注記ブロック：
  ```js
  if(state.reason === "II-no"){
    html += '<div class="result-note warn"><b>屋内での取り回しについて</b>自走式はハンドリムの分だけ幅を取り、屋内の狭い場所では小回りが利きにくい点にご注意ください。通路幅・回転スペースを事前にご確認いただくと安心です。</div>';
  }
  ```
  これはQ1-b-IIで「屋内は車椅子なしで移動できる」と答えたケース特有の注記で、今回追加する全結果共通の注記とは別物です。両方とも表示されて問題ありません（II-noの場合は2つ注記が並ぶことになりますが、内容が異なるため許容します）。

- 結果画面末尾の「この診断結果はあくまで参考です。ご心配な点があれば、福祉用具専門相談員にご相談ください。」の行は変更不要です。
