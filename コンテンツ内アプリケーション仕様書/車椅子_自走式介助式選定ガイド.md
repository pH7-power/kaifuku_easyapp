### 【車椅子選び】福祉用具専門相談員が教える「自走式」と「介助式」の選び方

「車椅子はとりあえず自走式を買っておけば安心」——そう考えて選んでいませんか？

実際、ネット通販で売れているのは自走式（自分でこぐタイプ）が中心です。しかし福祉用具専門相談員として現場でご相談を受けていると、実際にご本人が自分でこいで生活されているケースはごく一部で、多くのご家庭では「介助式」の方が結果的に満足度が高いというのが実感です。

この記事では、自走式と介助式の違いから、後悔しないための3つのチェックポイント、そしてよくある失敗パターンまで、購入前に知っておきたい情報をまとめました。

## 1. 基礎知識：自走式と介助式の徹底比較

|          | 自走式（自操式）           | 介助式         |
| -------- | ------------------ | ----------- |
| 後輪       | 大きい（ハンドリム付き）       | 小さい         |
| 操作       | 本人がこげる／介助者が押すことも可能 | 介助者が押す専用    |
| ブレーキ     | 本人用＋介助者用の場合あり      | 介助者の手元ブレーキ  |
| 重さ       | 重い（12〜14kg超が多い）    | 軽い（10kg前後〜） |
| 折りたたみ時の幅 | 広め・かさばる            | コンパクト       |
| 小回り      | やや苦手               | 得意          |

自走式は「自分でこげる」という自由度がある一方、後輪が大きい分どうしても重量・サイズが増します。介助式はその逆で、「軽さ・コンパクトさ・取り回しの良さ」に振り切った設計です。

## 2. 選定のための3大チェックポイント

### ① 本人の身体状況（本当にこげるか？）
- 上半身・腕にハンドリムを回し続けられる筋力があるか
- 認知機能に問題がなく、安全にブレーキ操作ができるか

※現実には、高齢の方でこの条件を満たし自走できる方はかなり稀です。「こげるかもしれないから」という期待だけで選ぶのは危険です。

### ② 使用環境（どこで使うか？）
- **屋内メイン**：自走式は部屋の角や廊下でつっかえやすく、コンパクトな介助式が有利
- **屋外メイン・段差が多い**：タイヤが大きい自走式の方が衝撃が少なく、介助者も段差を越えやすい

### ③ 介助者の状況（老々介護の視点）
- 介助者ご自身も高齢で、車椅子の積み下ろしに腰への負担を感じている
- 車のトランクへの積み下ろしが日常的に発生する

※「重量」は最重要チェック項目です。12〜14kg超の自走式を毎日積み下ろしするのは、腰痛の大きな原因になります。日々の積み下ろしがあるご家庭ほど、軽量な介助式のメリットは大きくなります。

## 3. 記事に必ず入れたい「よくある失敗・落とし穴」

- **「大は小を兼ねる」の罠**：「本人がこげるかもしれないから」と自走式を選んだものの、結局ほとんどこがず、「重くて狭い場所で使いにくいだけの車椅子」になってしまうケース
- **「車に載らない」問題**：軽自動車のトランクなど、事前に内寸を測っておかないと、自走式は折りたたんでも積み込めないことがあります
- **回転スペースの見落とし**：自宅の廊下やトイレ前の最小回転スペースを測らずに購入し、家の中で使えないケース

## 4. あなたはどっち？ YES/NOで分かる車椅子診断チェックリスト

以下の項目に、当てはまるものにチェックしてみてください。

- □ 本人は上半身・腕の筋力に自信がある
- □ 本人は自分でブレーキ操作を安全に行える
- □ 屋外での移動（段差・坂道）がメインである
- □ 介助者は車椅子の積み下ろしに負担を感じていない
- □ 車のトランク内寸を実測済みで、大きめの車椅子でも積める

**3つ以上チェックが付いた方 → 自走式（またはハイブリッド型）が候補**
**チェックが2つ以下の方 → 介助式がおすすめ**

多くのご家庭では、後者に当てはまるケースが実際には多いというのが現場の実感です。

## まとめ

自走式は「本人の自立」を後押しできる魅力的な選択肢ですが、実際にこげる方は限られており、多くのケースでは「軽さ・コンパクトさ・介助のしやすさ」を優先した介助式の方が、結果的に長く使われる車椅子になります。
